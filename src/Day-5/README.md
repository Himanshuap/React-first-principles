1. Context — Solves Prop Drilling
Prop drilling: threading a prop through many layers that don't use it, just to reach a deeply nested child. The middle components are forced delivery middlemen.
Context = a shared 'noticeboard' any component in the tree can read directly, without passing through every level. Props = hand a note person-to-person. Context = pin it to a board anyone can walk up and read.
